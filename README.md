## 🛆 Kurulum  


### 🛆 Adım 1: Bağımlılıkları Yükle  
```bash  
npm install
```  
### ⚙️ Adım 2: Yapılandırma  
1. `config.json` dosyasını açın.  
2. Discord API tokeninizi doldurun:  
```json  
{
    "token": "TOKENİNİZİ GİRİN",
    "sahip":"SAHİP IDSİ",
    "bot-adi":"Akemi Music"
}
```  
### 🚀 Adım 3: Botu Başlat  
```bash  
node index.js  
```

## 🛠️ Komutlar  
| Komut   | Açıklama                                     |  
|---------|---------------------------------------------|  
| `play` | Müzik çalar.    |  
| `skip` | Şarkıyı atlar.    |  
| `stop` | Müziği durdurur.    |  
| `queue` | Sırayı gösterir.    |
